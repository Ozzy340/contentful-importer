When something goes wrong mid-import, the team needs a clear runbook and a fast way to inspect the state files.

:::component CallToActionBanner
headline: Need help with a partial import?
body: Review the latest run state, inspect the failing document, and rerun safely after fixing the mapping.
:::

The phase-1 system keeps destructive actions out of scope so that rollback remains a controlled editorial decision.
